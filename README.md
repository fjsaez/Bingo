# Bingo![Bingo](/Imagen/icono_bingo2.png)![Pantalla principal](/Imagen/bingo_pantalla.jpg)---## DescripciónEl juego del Bingo (Bingo 75), donde se pueden registrar varios jugadores donde llenan los cartones y gana el que consiga completar determinados patrones en los mismos. Los  cartones son configurables. **Aún en fase de desarrollo.**---## Lenguaje/framework/base de datosDesarrollado en Delphi 1.2, usando Firemokey. Se emplearon sólo componentes nativos. Se usó Absolute Database edición personal como SGBD. ---## AutorIng. Francisco J.
